What makes it different

Designed for how audiobooks are actually built.

Most audiobook tools expect perfect input. m4Bookmaker is built around the reality of audiobook downloads — messy files, inconsistent formats, and titles that need work. It automatically detects and repairs corrupted audio before encoding, matches output settings to your source (bitrate, sample rate, channels), and gives you full chapter creation and inline editing with a built-in audio player — all without touching your originals.

Join a folder of files into a single clean audiobook

Drop any folder of audio files — MP3, M4A, AAC, FLAC, WAV, and more. File names become chapter titles automatically, track number prefixes are stripped, and everything is sorted and joined in order.

Set the title, author, narrator, and cover art. m4Bookmaker writes proper iTunes-compatible metadata so the audiobook shows up correctly in Books, Overcast, Audiobookshelf, and every major player.

  • Automatic chapter names from file names
  • Auto-selects the largest image as cover
  • Bitrate, sample rate, and mono/stereo matched to source
  • Mixed formats in the same folder — no problem

Open any .m4b file and rechapter it in Edit mode. Rename, reorder, merge, or remove chapter markers and save the changes back in-place — the audio stream is never touched. Every operation is non-destructive.

The built-in Split tool exports each chapter as a separate audio file, stream-copied and numbered automatically.

Start the queue and walk away. Jobs run sequentially. Failed jobs show an error and continue to the next — one bad source file won't derail the whole night's work.

Under the hood

Handles the mess so you don't have to.

Audio files from the internet are often damaged, inconsistent, or poorly formatted. m4Bookmaker detects and fixes all of it before the first frame is encoded.

  • 🔧
    Automatic MP3 repairCorrupted frames, missing VBR/Xing headers, embedded cover images — detected and fixed via stream-copy remux before encoding begins. No re-encoding, no quality loss.
  • 📊
    Preflight audio analysisEvery input file is analyzed with ffprobe. Sample rates, channel layouts, and bitrates are summarized. Encoding settings are matched to your source automatically.
  • 🎚
    Smart encoding match64k mono source → 64k mono output. 128k stereo → 128k stereo. The most common settings in your file set win. Override manually any time.
  • 🛡
    Non-destructive pipelineYour original files are never touched. Repaired copies go to a temp directory and are used only for that encode. Sources stay exactly as they were.
Command line

For power users and automation.

The CLI is a first-class citizen. Full flag and man page support. Use it in scripts, Docker, or CI pipelines.

# Build an audiobook from a folder
$ m4bmaker ./Dune \
    --title    "Dune" \
    --author   "Frank Herbert" \
    --narrator "Scott Brick" \
    --cover    cover.jpg

# Batch-build an entire audiobook library
$ for dir in ~/Audiobooks/*/; do
    m4bmaker "$dir" --output ~/Output/
  done
